## IPA Quiz - Interactive Phonetic Symbol Learning Tool

An interactive web-based quiz application that helps students learn International Phonetic Alphabet (IPA) symbols through audio recognition.

**Features:**
- Audio-based learning with real IPA sound samples
- Multiple categories: vowels, consonants, or all symbols
- Interactive multiple-choice quiz with instant feedback
- Score tracking and progress monitoring
- Responsive design for desktop and mobile
