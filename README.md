# Phakphum's Resume Website

## Introduction
This repository holds my unique resume website, a tribute to the classic Windows 95 interface with a twist of modern web development. It serves as a fun and interactive way to learn about my professional background in software engineering.

## Special Thanks
A heartfelt thank you to the creator of the original [Windows 95 UI Kit](https://github.com/DonChiaQE/win95) on GitHub. Their work inspired the aesthetic and functionality that make this site both nostalgic and engaging.

## Features
- **Classic Windows 95 Theme:** A creative and engaging way to present my professional profile.
- **Interactive Experience:** Users can interact with desktop elements reminiscent of the classic Windows 95 interface.
- **Ongoing Development:** A dedicated space indicating the ongoing addition of new content and features.

## Technologies
- **Vue.js:** For building a dynamic and responsive user interface.
- **Vuex:** For efficient state management throughout the application.
- **Classic Styling:** Using CSS and HTML to recreate the Windows 95 look and feel.

## Getting Started
To set up this project on your local machine:
1. Clone the repository.
2. Follow the setup instructions specified in the [Windows 95 UI Kit Readme](https://github.com/DonChiaQE/win95/blob/main/README.md).
